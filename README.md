# 🌍 LocalConnect

LocalConnect is a Flask-based web application that helps users discover local roadside shops and small businesses in Nellore. It provides shop details, contact information, Google Maps location, and category-wise browsing in a simple and user-friendly interface.

---

## ✨ Features

- 🔍 Search shops
- 🏪 Category-wise shop listing
- 📍 Google Maps integration
- 📞 Click-to-call feature
- 🌙 Dark Mode
- 📱 Mobile Responsive Design
- ⬆️ Back to Top button
- 📄 Individual shop pages
- 📷 Shop image gallery

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
